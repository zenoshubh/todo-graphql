# GraphQL Todo App

This is a full-stack Todo application built with GraphQL, Apollo Server, and Apollo Client. The application demonstrates the integration of modern web technologies for building a complete todo management solution.

## Key Technologies

### Backend
- **GraphQL**: Schema-driven API development with types and operations
- **Apollo Server**: GraphQL server implementation
- **MongoDB**: Database for storing users and todos
- **JWT Authentication**: Secure user authentication
- **TypeScript**: Type-safe backend development

### Frontend
- **React**: User interface development
- **Apollo Client**: State management and data fetching
- **Tailwind CSS**: Utility-first styling
- **TypeScript**: Type-safe frontend development

## GraphQL Features

- **Queries**: Data fetching operations to retrieve information
- **Mutations**: Data manipulation operations to create, update, and delete records
- **Resolvers**: Backend functions that fulfill GraphQL operations
- **Schema**: Type definitions that shape the data and available operations

## Project Status

This project is currently under development and in its early stages.
